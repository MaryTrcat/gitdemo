# 初步系统设计

## 目标

构建一个 Qt + C++ + MySQL 的手语动作学习系统初版，满足八个功能模块展示和后续扩展到真实计算机视觉检测的需要。

## 八模块划分

- 用户管理：注册、登录、当前用户状态。
- 手势资源库：手势列表、分类、难度、说明。
- 摄像头采集：摄像头启动入口与状态显示。
- 关键点识别：统一 `HandKeypointDetector` 接口，当前使用模拟 21 点。
- 动作评分：`GestureScorer` 计算位置、角度、稳定性与总分。
- 纠错反馈：按评分生成中文反馈。
- 学习统计：查询练习次数、平均分和最近练习。
- 考核评测：随机选择手势练习，低分写入错题本。

## 数据流

用户登录后选择手势，系统启动练习。摄像头模块提供图像帧，关键点检测模块输出 21 点，评分模块与标准关键点比对生成分数，反馈模块生成建议，最后写入 MySQL 的练习记录和错题本。

## 初版取舍

初版重点是系统闭环和 C++ 工程结构。真实 MediaPipe/OpenCV 检测先以接口形式预留，避免早期卡在第三方库集成上。
